Importar / Exportar Datos

¿Cómo funciona?

Este módulo te permite respaldar y restaurar todos los datos de tu tienda (productos, categorías, contenido, redes sociales) mediante un archivo JSON.

✓ Se exporta:

  • • Productos (título, precio, descripción, SKU, estado)
  • • Categorías (se infieren de los productos)
  • • Contenido (Sobre nosotros, Ubicación, páginas legales)
  • • Configuración de Firebase
  • • Credenciales de admin
  • • Redes sociales

✗ No se exporta:

  • • Imágenes (se quedan en Cloudinary)

Las URLs de las imágenes sí se incluyen pero las imágenes físicas no.

Analytics & SEO

Conecta tu tienda con servicios de analítica, búsqueda y publicidad. Los scripts se inyectan automáticamente en el front.

¿Para qué sirve esto?

Google Analytics: Para medir visitas y comportamiento de usuarios. Google Search Console: Para monitorear tu posicionamiento en Google. Meta Pixel (Facebook): Para trackear conversiones y crear audiencias para anuncios.

📊 Google Analytics

Mide tus visitas y el comportamiento de tus usuarios

Formato: G-XXXXXXXXXX. Lo encuentras en Admin > Flujos de datos > ID de medición.

Si pegas el código completo, este tendrá prioridad sobre el ID de arriba.

🔍 Google Search Console

Verifica tu sitio en Google y monitorea el SEO

Pega solo el valor del atributo content="..." del meta tag de verificación HTML.

📘 Meta Pixel (Facebook)

Trackea conversiones y crea audiencias para anuncios

Pixel ID numérico de 15-16 dígitos. Lo encuentras en Events Manager > Data Sources.

📐 Medidas Recomendadas

  • Tamaño ideal: 800 × 1000 px (ratio 4:5 vertical)
